# Restaurant Ordering System
## Overview

This project is a restaurant ordering system that demonstrates object-oriented programming concepts including encapsulation, inheritance, polymorphism, and composition.



## How to Run: 

1. Make sure Python 3 is installed
2. Open the project folder
3. Run the program: python main.py



## Project structure:

    |__ report.md
 


## Example Commands: 

- This project runs a predefined scenario in main.py
- The following operations are demonstrated:
- Creating an order
- Adding items to an order
- Applying discounts
- Confirming an order
- Changing order status
- Printing a receipt
